## 返利计提 SOP

### 计提原则
```
1. 权责发生制：销售发生时即计提，不等实际到账
2. 谨慎性原则：只计提有明确商务政策文件支撑的返利
3. 分品牌独立：每个品牌的返利独立计提，不跨品牌冲抵
4. 逐车匹配：返利与车辆 VIN 一一对应，形成返利台账
```

### 每月计提流程

```
Step 1: 获取商务政策
  → 各品牌主机厂每月/每季下发商务政策文件
  → 财务虾解析政策条款，提取返利计算公式

Step 2: 匹配销售数据
  → 从 DMS 导出当月零售/批发数据（含 VIN/车型/开票日期/销售顾问）
  → 按政策条款逐台匹配返利标准

Step 3: 计算计提金额
  → 基础返利 = Σ(每台车返利标准 × 达成系数)
  → 满意度返利 = 季度考核得分 × 系数 × 销量
  → 专项返利 = 按专项政策单独计算

Step 4: 编制计提凭证
  → 借：应收账款 - 应收返利款 - XX 品牌
  → 贷：主营业务收入 - 返利收入

Step 5: 到账核销
  → 主机厂回款后核销应收返利款
  → 差异分析：到账金额 vs 计提金额，标注原因
```

### 返利差异分析

| 差异类型 | 常见原因 | 处理方式 |
|----------|----------|----------|
| 满意度扣减 | 密采/J.D.Power 未达标 | 分析扣分项，反馈业务部门整改 |
| 销量阶梯差 | 差几台进入下一阶梯 | 月末冲量成本 vs 返利增量测算 |
| 申报漏报 | 未按时提交返利申报材料 | 建立申报日历，设置截止提醒 |
| 政策解读偏差 | 商务政策条款理解有误 | 联系主机厂区域经理确认 |

> 返利是 4S 店核心利润来源，新车裸车普遍倒挂，返利决定盈亏。
