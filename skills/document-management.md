# 公文管理技能

## 技能定位
本技能用于集团行政公文的起草、审核、流转、归档全流程管理，确保公文格式规范、流程合规、存档完整。

## 支持场景
| 场景 | 说明 |
|------|------|
| 通知/通告 | 发布集团重大事项、制度变更、活动通知 |
| 请示/批复 | 下级向上级请示事项的流转与批复 |
| 会议纪要 | 集团级会议的纪要撰写与分发 |
| 通报 | 表彰、批评、情况通报 |
| 函件 | 对外公函（政府/供应商/合作伙伴） |
| 报告 | 工作报告、调研报告、总结报告 |
| 决议/决定 | 集团重要决策的正式发文 |

## 公文格式规范

### 发文字号
- 格式: 铠祺〔20XX〕XX号
- 例: 铠祺〔2026〕15号

### 公文结构
1. **标题**: 方正小标宋简体，二号，居中
2. **主送机关**: 仿宋，三号，顶格
3. **正文**: 仿宋，三号，首行缩进2字符
4. **附件说明**: 仿宋，三号
5. **发文机关署名**: 仿宋，三号，右对齐
6. **成文日期**: 仿宋，三号，右对齐
7. **印章**: 居中压日期

### 版记要素
- 抄送机关
- 印发机关
- 印发日期
- 共印份数

## 公文流转流程

1. **起草**: 承办部门起草 → 部门负责人审核
2. **核稿**: 行政中心对格式、内容合规性核查
3. **会签**: 涉及多部门的公文需相关部门会签
4. **签发**: 按权限报领导签发
5. **编号**: 行政中心统一编制发文字号
6. **印发**: 制作正式文件并加盖印章
7. **分发**: 按主送/抄送范围分发
8. **归档**: 电子版+纸质版存档备查
