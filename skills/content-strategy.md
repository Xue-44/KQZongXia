# 内容策略技能 — 集团综合版

> **版本**: v2.0 | **更新日期**: 2026-05-23 | **适用 Agent**: 总虾 (xia-zong)
> **来源**: 市场虾 + 新媒体虾 合并去重

## 技能定位

为铠祺集团旗下四品牌（广汽传祺、上汽奥迪、广汽昊铂、广汽埃安）制定全域内容策略，覆盖市场推广内容策划和新媒体内容运营两大领域，确保品牌调性统一、投放精准、转化可追踪。

## 适用场景

本技能面向集团7个部门统一提供内容策略支持：
- 市场部：月度/季度推广内容规划、新车上市内容策划、跨渠道预算分配
- 新媒体部：月度/周度选题策划、热点捕捉、爆款方法论
- 销售部：促销内容策划、车型卖点梳理
- 售后部：车主关怀内容、服务推广内容
- 财务部：费用预算内容模板
- 客服部：客户沟通话术内容
- 行政部：内部宣传内容规范

## 标准流程

1. **市场环境扫描**：收集行业动态、区域市场特征、消费者趋势
2. **热点扫描**：微博热搜、抖音热榜、小红书趋势（每日 09:00 / 14:00 / 19:00）
3. **品牌定位校准**：结合品牌战略确认当期内容调性和核心信息
4. **受众分析**：结合各品牌目标受众画像和内容消费偏好
5. **渠道策略制定**：根据目标受众画像选择投放渠道组合
6. **内容日历输出**：制定详细排期（内容形式+渠道+预算+预期KPI）
7. **Brief输出**：详细内容Brief（标题、脚本大纲、视觉方向、BGM建议）
8. **投放执行跟踪**：按节点跟踪投放进度，监控各渠道效果
9. **数据追踪**：发布后72小时数据跟踪，及时调整策略
10. **效果复盘优化**：基于数据调整下一周期策略

## 四品牌内容定位与策略

### 广汽传祺
- **定位**：家庭实用 + 性价比 + 空间舒适
- **核心受众**：30-45岁家庭用户、中小企业主、三四线城市中产
- **内容调性**：务实可靠、温馨生活、精打细算，接地气不low
- **核心卖点**：大空间/7座刚需（M8/M6系列MPV）、同价位配置优势、低使用成本、真实口碑
- **投放渠道**：汽车之家、懂车帝（核心）+ 抖音本地推（辅助）
- **内容形式**：图文测评40% + 短视频35% + 软文15% + 硬广10%
- **发布节奏**：每日2-3条，集中在12:00/18:00/20:00
- **预算分配建议**：垂媒60% + 短视频30% + 其他10%
- **KPI指标**：线索量、到店率、成交转化率

### 上汽奥迪
- **定位**：豪华尊贵 + 品牌调性 + 圈层营销
- **核心受众**：35-50岁高净值人群、企业主、政商精英
- **内容调性**：高级质感、圈层归属、低调奢华，高级不冰冷
- **核心卖点**：品牌溢价与身份象征、灯厂科技与quattro四驱、专属服务与私享体验
- **投放渠道**：高端生活方式平台（小红书/知乎）+ 机场/高尔夫/高端商圈户外
- **内容形式**：圈层活动图文50% + 品牌视频25% + 软文/专访15% + 硬广10%
- **发布节奏**：每日1-2条，集中在19:00/21:00
- **预算分配建议**：圈层活动40% + 户外媒体25% + 数字内容25% + 其他10%
- **KPI指标**：潜客线索质量、试驾预约、圈层触达人数、品牌NPS

### 广汽昊铂
- **定位**：智能科技 + 高端新能源 + 技术领先
- **核心受众**：28-40岁科技新贵、新能源早期采用者、极客人群
- **内容调性**：技术硬核、前卫设计、极致体验，前卫不浮夸
- **核心卖点**：智能驾驶（城市NOA/高速NOA）、超充技术（800V平台）、极致性能、科技美学
- **投放渠道**：科技媒体（36氪/虎嗅/汽车之心）+ B站（核心）+ 知乎
- **内容形式**：深度技术解读40% + 体验视频30% + 图文科普20% + 硬广10%
- **发布节奏**：每日2-3条，集中在12:00/18:00/21:00
- **预算分配建议**：B站创作者合作35% + 科技媒体25% + 体验活动25% + 其他15%
- **KPI指标**：技术内容传播量、品牌科技认知度、试驾转化率、社群活跃度

### 广汽埃安
- **定位**：年轻时尚 + 环保 + 智能互联
- **核心受众**：25-35岁年轻家庭、首购用户、科技潮人
- **内容调性**：潮酷活力、智能有趣、环保态度，潮酷不低幼
- **核心卖点**：智能座舱（车机生态/语音交互）、潮改文化、弹匣电池/夸克电驱技术普惠、高性价比
- **投放渠道**：抖音（核心）+ 小红书（核心）+ 微信视频号
- **内容形式**：短视频45% + 图文种草30% + 直播15% + 硬广10%
- **发布节奏**：每日2-3条，集中在12:00/17:00/20:00
- **预算分配建议**：抖音信息流40% + 小红书达人30% + 直播运营20% + 其他10%
- **KPI指标**：内容曝光量、互动率、线索留资、到店转化

## 内容形式规范

| 形式 | 适用品牌 | 时长/篇幅 | 核心要点 |
|------|---------|----------|---------|
| 图文测评 | 传祺/奥迪 | 1500-3000字 | 专业参数+真实体验+竞品对比 |
| 短视频 | 埃安/昊铂 | 15-60秒 | 前3秒钩子+节奏紧凑+结尾引导 |
| 直播 | 埃安/传祺 | 60-120分钟 | 产品讲解+实时互动+留资闭环 |
| 软文 | 奥迪/昊铂 | 2000-4000字 | 深度内容+品牌故事+圈层共鸣 |
| 硬广 | 全品牌 | 按媒体规格 | 品牌信息+核心卖点+行动号召 |

## 爆款内容公式（新媒体虾贡献）

```
高完播率 = 前3秒钩子 + 节奏紧凑 + 音乐卡点 + 结尾引导
高互动率 = 争议性话题 + 提问式结尾 + 评论区运营
高转发率 = 实用干货 + 情感共鸣 + 社交货币
```

## 投放策略框架

### 渠道选择矩阵
```
                高潜客精准度
                    
        昊铂科技媒体 | 奥迪圈层渠道
        (深度内容)   | (私享活动)
                    
    ────────────────┼────────────────→ 高曝光广度
                    
        传祺垂媒    | 埃安抖音/小红书
        (性价比传播) | (流量种草)
                    
                低潜客精准度
```

### 预算分配原则

| 档位 | 月预算范围 | 渠道配比 | 适用阶段 |
|------|----------|---------|---------|
| S级 | 50万+ | 内容40%+投放35%+活动25% | 新车上市/大型活动 |
| A级 | 20-50万 | 内容35%+投放40%+活动25% | 旺季冲量/季度冲刺 |
| B级 | 10-20万 | 内容45%+投放35%+活动20% | 常规运营 |
| C级 | <10万 | 内容50%+投放30%+活动20% | 淡季维护 |

### 效果预估模型

- **曝光预估**：预算 x 渠道CPM x 内容质量系数（0.8-1.2）
- **线索预估**：曝光量 x 渠道线索率（0.3%-2%）
- **到店预估**：线索量 x 到店转化率（15%-30%）
- **成交预估**：到店量 x 成交率（10%-25%）

## 热点追踪与借势机制

- **日常监控**：微博热搜、抖音热榜、小红书趋势（每日 09:00 / 14:00 / 19:00）
- **行业热点**：汽车之家/懂车帝 首页推荐、新车发布日历（每日 10:00）
- **竞品动态**：区域竞品促销/活动自动抓取
- **借势时效**：热点发生 2 小时内完成选题判断，4 小时内出内容初稿
