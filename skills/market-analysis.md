# 市场分析技能

## 适用场景
- 各品牌月度/季度/年度市场表现分析
- 区域市场竞争力评估与份额变化追踪
- 竞品市场策略深度解析与应对建议
- 市场趋势预判与机会点识别

## 标准流程
1. **数据采集**：销售数据、市场费用、线索数据、竞品情报
2. **多维度分析**：品牌/车型/区域/时间四维交叉分析
3. **对标评估**：与竞品对比、与历史对比、与目标对比
4. **问题诊断**：识别增长瓶颈、费用低效点、策略偏差
5. **策略输出**：基于数据的优化建议和行动计划

## 核心分析框架

### 1. 销售表现分析
| 维度 | 分析指标 | 健康标准 |
|------|---------|---------|
| 总量 | 订单量、交付量、线索量 | 环比增长 >0% |
| 结构 | 车型销量结构、价格带分布 | 主力车型占比 >40% |
| 趋势 | 月度趋势、季节性波动 | 符合行业规律 |
| 目标 | 目标达成率、缺口分析 | 月度达成率 >80% |

### 2. 市场费用分析
| 维度 | 分析指标 | 健康标准 |
|------|---------|---------|
| 总量 | 总费用、费效比（费用/销量） | 费效比 < 行业均值 |
| 结构 | 费用类型分布（广告/活动/物料） | 广告费占比 <50% |
| 效率 | 单线索成本、单订单成本 | 环比下降或持平 |
| 预算 | 预算执行率、超支预警 | 月度执行率 80-120% |

### 3. 竞品对标分析
| 维度 | 分析指标 | 数据来源 |
|------|---------|---------|
| 价格 | 成交价、优惠幅度、金融政策 | 市场调研、竞品官网 |
| 促销 | 活动频次、力度、形式 | 竞品广告、门店走访 |
| 渠道 | 经销商数量、位置分布 | 地图搜索、行业报告 |
| 声量 | 广告投放量、媒体露出度 | 广告监测、舆情监测 |

### 4. 区域市场分析
| 维度 | 分析指标 | 决策依据 |
|------|---------|---------|
| 份额 | 区域市场份额、排名变化 | 资源倾斜优先级 |
| 潜力 | 人口、经济、竞品覆盖 | 新店选址建议 |
| 效率 | 单店产出、人均效能 | 团队优化方向 |
| 问题 | 薄弱区域识别、原因分析 | 专项扶持计划 |

## 分析方法库

### 趋势分析法
- **移动平均**：7日/30日滚动均值，平滑波动
- **同比环比**：与去年同期/上期对比，识别趋势
- **季节性分解**：分离趋势、季节、残差，预测未来

### 结构分析法
- **帕累托分析**：80/20 法则，识别核心贡献车型
- **波士顿矩阵**：明星/现金牛/问题/瘦狗，产品组合策略
- **价格带分析**：各价格段市场份额，定价策略优化

### 对标分析法
- **雷达图对标**：多维度综合对比（价格/促销/渠道/服务）
- **SWOT 分析**：优势/劣势/机会/威胁，战略定位
- **竞争态势矩阵**：市场份额 vs 增长率，市场地位评估

## 异常诊断模型

### 销量异常诊断树
```
销量下降
├─ 线索量下降（前端问题）
│  ├─ 广告投放减少
│  ├─ 竞品促销冲击
│  └─ 市场热度下降
├─ 线索转化率下降（中端问题）
│  ├─ 销售话术问题
│  ├─ 价格竞争力不足
│  └─ 库存匹配度低
└─ 交付量下降（后端问题）
   ├─ 供应链问题
   ├─ 金融审批收紧
   └─ 客户流失竞品
```

### 费用异常诊断树
```
费用超支
├─ 结构性超支
│  ├─ 广告费占比过高
│  ├─ 活动费效率低下
│  └─ 物料费浪费严重
├─ 执行性超支
│  ├─ 单项目超预算
│  ├─ 审批流程失效
│  └─ 供应商报价虚高
└─ 策略性超支
   ├─ 市场环境变化
   ├─ 竞争加剧
   └─ 战略调整
```

## 报告输出格式

### 月度市场分析报告（结构）
```
1. 执行摘要（一页核心结论）
2. 市场环境概览
   - 行业整体趋势
   - 竞品动态汇总
3. 各品牌表现深度分析
   - 销售数据（订单/交付/线索）
   - 市场费用（总量/结构/效率）
   - 库存状况（深度/周转）
4. 区域市场表现
   - 各区域份额与排名
   - 重点区域专项分析
5. 问题诊断与机会识别
   - 核心问题（不超过3个）
   - 潜在机会（不超过3个）
6. 下月策略建议
   - 销售目标调整建议
   - 市场费用分配建议
   - 重点行动项
```

### 专项分析报告类型
| 报告类型 | 触发条件 | 输出频率 |
|---------|---------|---------|
| 竞品深度分析 | 竞品发布新车/重大促销 | 事件驱动 |
| 区域市场评估 | 新店选址/区域调整 | 项目驱动 |
| 费用效率分析 | 费效比连续2月恶化 | 月度 |
| 车型生命周期分析 | 车型上市满1年 | 年度 |
