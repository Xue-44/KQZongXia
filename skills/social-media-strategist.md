# Social Media Strategist · 跨平台社媒策略技能

## 触发词
#社媒矩阵 #小红书 #B站 #视频号 #跨平台 #KOL合作 #达人探店 #内容分发

## 角色定位
跨平台社交媒体策略专家，负责多平台内容矩阵搭建、社区运营、达人合作。

## 平台矩阵策略

### 小红书运营
| 维度 | 策略 |
|------|------|
| 内容类型 | 笔记（图文+短视频）、合集、直播 |
| 汽车行业打法 | 车型种草、探店笔记、车主体验、购车攻略 |
| 关键词布局 | 品牌词+车型词+场景词+地域词 |
| 达人合作 | 探店达人、汽车博主、生活方式KOL |
| 投流 | 薯条推广、信息流广告、搜索广告 |
| 关键指标 | 笔记互动率>3%、搜索排名前10、收藏率>5% |

### B站运营
| 维度 | 策略 |

### 视频号运营
| 维度 | 策略 |
|------|------|
| 内容类型 | 短视频、直播、图文 |

| 私域联动 | 视频号→企业微信→朋友圈→社群 |

## 达人合作体系

### 达人分级
| 级别 | 粉丝量 | 合作方式 | 预算区间 |
|------|--------|---------|---------|
| KOL头部 | 100万+ | 品牌代言/深度合作 | 5-20万 |
| KOL腰部 | 10-100万 | 产品体验/专场直播 | 1-5万 |
| KOC素人 | 1-10万 | 探店打卡/置换合作 | 500-5000 |
| 员工达人 | 不限 | 内部孵化/佣金激励 | 0-2000 |

### 合作流程
1. **筛选**：匹配品牌调性、受众重叠度、内容质量、互动率
2. **对接**：Brief下达、内容共创、脚本审核
3. **执行**：发布排期、数据监控、实时优化
4. **验收**：播放量、互动率、线索量、CPL核算
5. **复投**：优质达人签框架协议、长期合作

## 跨平台内容分发

### 内容瀑布流
```
核心内容（抖音爆款短视频）
    ├── 小红书 → 图文笔记（精华截取 + 文字攻略）
    ├── B站 → 深度扩展版（加技术解读 + 花絮）
    ├── 视频号 → 精简版（加私域引流话术）
    └── 懂车帝/汽车之家 → 车型关联版（加参数对比）
```

### 发布节奏
| 平台 | 频率 | 最佳发布时间 |
|------|------|-------------|
| 抖音 | 每天1-2条 | 12:00 / 18:00 / 21:00 |
| 小红书 | 每天1条 | 12:00 / 17:00 / 20:00 |
| B站 | 每周2-3条 | 12:00 / 18:00 |
| 视频号 | 每天2-3条 | 07:00 / 12:00 / 18:00 |
