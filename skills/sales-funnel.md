# 销售漏斗管理技能

## 技能定位
从线索到交车的全链路销售漏斗管理，追踪各阶段转化率，发现瓶颈环节，提出改进方案。

## 漏斗阶段定义

| 阶段 | 定义 | 关键指标 |
|------|------|---------|
| 线索（Lead） | 获取客户联系方式 | 线索量、线索来源分布 |
| 有效线索 | 接通且有意向 | 有效率 |
| 到店（Walk-in） | 客户到店看车 | 到店率 = 到店/有效线索 |
| 试驾（Test Drive） | 客户完成试驾 | 试驾率 = 试驾/到店 |
| 订单（Order） | 签署合同付定金 | 订单转化率 = 订单/试驾 |
| 交车（Delivery） | 完成车辆交付 | 交车率 = 交车/订单 |
| 战败（Lost） | 客户流失/选择竞品 | 战败率、战败原因分布 |

## 核心公式

- **整体转化率** = 交车数 / 线索总量
- **有效率** = 有效线索 / 线索总量
- **到店率** = 到店数 / 有效线索
- **试驾率** = 试驾数 / 到店数
- **订单转化率** = 订单数 / 试驾数
- **交车率** = 交车数 / 订单数
- **库销比** = 库存量 / 月均销量

## 全品牌漏斗标准

| 转化节点 | 传祺标准 | 奥迪标准 | 昊铂标准 | 埃安标准 |
|---------|---------|---------|---------|---------|
| 有效率 | ≥60% | ≥50% | ≥55% | ≥55% |
| 到店率 | ≥35% | ≥25% | ≥30% | ≥30% |
| 试驾率 | ≥60% | ≥55% | ≥50% | ≥50% |
| 订单转化率 | ≥30% | ≥20% | ≥25% | ≥25% |
| 交车率 | ≥85% | ≥85% | ≥80% | ≥80% |
| 整体转化率 | ≥5% | ≥3% | ≥3% | ≥3% |

## 瓶颈分析与改进

- **有效率低** → 线索质量差，优化投放渠道和话术
- **到店率低** → 邀约话术不足，加强跟进和利益点包装
- **试驾率低** → 展厅体验差或产品介绍不到位，强化试驾体验
- **订单转化率低** → 价格谈判能力弱，加强议价培训和促销政策
- **交车率低** → 金融/保险/上牌流程慢，优化后端交付效率

## 战败分析维度

- 价格因素（竞品更便宜/预算不足）
- 产品因素（配置不满意/外观不喜欢）
- 服务因素（销售顾问态度/专业度）
- 竞品因素（选择了某个具体竞品）
- 其他（家庭意见不统一/等待新款）

## 使用方式

```
使用 sales-funnel 技能分析 [品牌/门店] [时间段] 的销售漏斗：
- 提供各阶段数据
- 标注瓶颈环节
- 给出改进建议
```
