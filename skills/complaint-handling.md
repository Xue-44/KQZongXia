# 投诉处理

客户投诉分级处理与化解

## 投诉严重等级

| 等级 | 定义 | 示例 | 响应时限 | 处理人 |
|------|------|------|----------|--------|
| **L1 轻微** | 一般不满，可快速化解 | 等待时间略长、销售回复慢 | 2 小时 | DCC 专员 |
| **L2 一般** | 明确不满，需补偿安抚 | 交车延迟、配置与宣传不符 | 4 小时 | DCC 主管 |
| **L3 严重** | 强烈不满，涉及经济诉求 | 车辆质量问题、金融纠纷 | 8 小时 | 客服总监+品牌总经理 |
| **L4 重大** | 可能引发舆情/法律风险 | 安全事故、欺诈、集体投诉 | 1 小时 | 集团总经理+法务 |
| **L5 危机** | 已引发舆情/监管部门介入 | 媒体曝光、315 投诉、诉讼 | 立即 | 集团应急小组 |

## 投诉处理 SOP

### 通用流程

```
1. 倾听 → 完整听完客户诉求，不打断、不辩解
2. 共情 → "我非常理解您的心情，换作是我也会很着急"
3. 确认 → 复述客户问题，确认理解准确
4. 方案 → 给出具体解决时间线和补偿方案
5. 执行 → 协调内部资源，按时完成解决
6. 闭环 → 解决后 24 小时内回访确认满意度
```

## 话术金句库

### 安抚类

| 场景 | 话术 |
|------|------|
| 客户情绪激动 | "X先生，我完全理解您的感受，请您先消消气，我现在的任务就是把您的问题彻底解决" |
| 承认不足 | "这件事确实是我们的疏忽，我们没有做好，我代表铠祺向您道歉" |
| 需要调查 | "您说的这个情况我需要马上核实，给我 [时间] 时间，我第一时间给您回复" |
| 客户要求赔偿 | "您的诉求我听到了，我先确认解决方案，一定给您一个满意的答复" |

### 化解类

| 场景 | 话术 |
|------|------|
| 交车延迟 | "目前您的车已在 [阶段]，为了确保交付质量，我们多花了几天做最终检查，[日期] 一定交付" |
| 价格不满 | "价格虽然没法再降了，但我帮您争取了一套 [精品/保养]，价值 [金额]" |
| 车辆小瑕疵 | "这个小问题我们售后马上处理，再额外送您一次全车精洗 + 保养代金券" |
