# 单车毛利分析

## 技能名称
单车毛利三维分析（裸车毛利 / 水平业务毛利 / 综合毛利）

## 适用场景

| 场景 | 触发词 | 输出 |
|------|--------|------|
| 单车毛利表 | "单车毛利""一台车赚多少" | 单车毛利明细表 |
| 车型毛利对比 | "哪个车型毛利高""车型利润" | 车型毛利排名 |
| 水平业务毛利 | "金融赚多少""保险利润""精品毛利" | 水平业务拆分 |
| 价格异常分析 | "卖亏了""负毛利""倒挂" | 负毛利车辆清单 |
| 销售顾问毛利 | "谁卖车最赚钱""顾问毛利排名" | 顾问毛利排名表 |

## 三维毛利模型

```
单车综合毛利 = 裸车毛利 + 水平业务毛利

裸车毛利 = 实际开票价 - 采购成本（含运费） + 返利分摊
水平业务毛利 = 金融佣金 + 保险佣金 + 精品毛利 + 上牌服务 + 延保收入 + 其他增值

裸车毛利率 = 裸车毛利 / 开票价
综合毛利率 = 综合毛利 / (开票价 + 水平业务收入)
```

## 负毛利车辆分析

| 风险等级 | 条件 | 处理方式 |
|----------|------|----------|
| 🔴 严重倒挂 | 综合毛利 < -5,000 | 逐台复盘，销售总监说明原因 |
| 🟡 轻微倒挂 | 综合毛利 -5,000 ~ 0 | 关注趋势，月末批量分析 |
| 🟢 正常盈利 | 综合毛利 > 0 | — |

### 负毛利常见原因

| 原因 | 占比估计 | 是否合理 |
|------|----------|----------|
| 月末冲量达标返利阶梯 | 40% | ✅ 合理（需测算跨度价值） |
| 长库龄车清库 | 25% | ✅ 合理（但需控制超期比例） |
| 竞争比价被迫让价 | 20% | ⚠️ 关注趋势 |
| 价格审批漏洞 | 10% | ❌ 需整改 |
| 计算错误/政策未用足 | 5% | ❌ 需培训 |
