# 费用管控技能

## 适用场景
- 各品牌市场费用预算编制与分配
- 费用审批流程监控与异常预警
- 费用核销进度追踪与逾期处理
- 费效比分析与优化建议
- 商务政策返利计算与追踪

## 标准流程
1. **预算编制**：年度/季度/月度预算分解到品牌/项目
2. **费用审批**：监控审批流程，识别异常（超预算/超标准）
3. **执行监控**：实时追踪费用支出，预警超支风险
4. **核销管理**：督促及时核销，处理逾期未核销
5. **分析优化**：费效比分析，提出优化建议

## 预算管理体系

### 预算层级结构
```
年度总预算（集团层面）
├─ 品牌维度预算（传祺/奥迪/昊铂/埃安）
│  ├─ 费用类型预算（广告费/活动费/物料费/其他）
│  │  ├─ 项目维度预算（具体活动/广告投放）
│  │  └─ 月度分解预算（1月-12月）
│  └─ 区域维度预算（银川/石嘴山/吴忠等）
└─ 备用金预算（应急/突发）
```

### 预算编制原则
| 原则 | 具体规则 | 示例 |
|------|---------|------|
| 历史基准 | 参考去年同期实际支出 ±10% | 去年Q1广告费100万，今年预算110万 |
| 目标驱动 | 按销售目标比例分配 | 每100台销量配10万市场费 |
| 竞争对标 | 参考竞品投入水平 | 竞品单店月均投入5万，我方预算6万 |
| 战略倾斜 | 向重点品牌/车型倾斜 | 昊铂新品上市，预算上浮30% |

### 预算审批流程
```
申请 → 部门审核 → 财务审核 → 管理层审批 → 执行
   ↓        ↓         ↓          ↓         ↓
 预算检查 标准检查 合规检查  战略匹配 资金预留
```

## 费用类型与管控标准

### 广告费（占比 30-50%）
| 子类 | 管控标准 | 核销要求 |
|------|---------|---------|
| 线上广告 | CPL ≤ 行业均值1.2倍 | 平台数据截图 + 线索明细 |
| 线下广告 | 单点位月费 ≤ 市场价 | 发布照片 + 位置证明 |
| 媒体合作 | 按刊例价 8 折以内 | 发布证明 + 效果报告 |
| KOL 合作 | ROI ≥ 1.5 | 内容链接 + 互动数据 |

### 活动费（占比 20-40%）
| 子类 | 管控标准 | 核销要求 |
|------|---------|---------|
| 店头活动 | 单场费用 ≤ 1万元 | 活动方案 + 现场照片 + 签到表 |
| 外展活动 | 单场费用 ≤ 3万元 | 场地合同 + 搭建照片 + 客户登记 |
| 试驾活动 | 人均成本 ≤ 200元 | 试驾协议 + 反馈表 + 线索转化 |
| 车主活动 | 人均成本 ≤ 150元 | 活动方案 + 参与名单 + 满意度调查 |

### 物料费（占比 10-20%）
| 子类 | 管控标准 | 核销要求 |
|------|---------|---------|
| 印刷品 | 单价 ≤ 市场价1.1倍 | 印刷合同 + 验收单 |
| 礼品 | 单件成本 ≤ 50元 | 采购合同 + 入库单 + 领用记录 |
| 展具 | 按折旧年限分摊 | 资产卡片 + 使用记录 |
| 服装 | 按岗位标准配置 | 采购清单 + 发放记录 |

### 其他费用（占比 5-15%）
| 子类 | 管控标准 | 核销要求 |
|------|---------|---------|
| 差旅费 | 按集团差旅标准 | 出差申请 + 票据 |
| 培训费 | 人均 ≤ 500元/天 | 培训通知 + 签到表 |
| 咨询费 | 事前审批 | 合同 + 交付物 |
| 软件费 | 按年度预算 | 合同 + 使用证明 |

## 费用核销管理

### 核销时效要求
| 费用类型 | 核销期限 | 逾期处理 |
|---------|---------|---------|
| 广告费 | 活动结束后 7 日内 | 每超1天扣责任人绩效 1分 |
| 活动费 | 活动结束后 3 日内 | 每超1天扣责任人绩效 2分 |
| 物料费 | 收货后 2 日内 | 每超1天扣责任人绩效 1分 |
| 差旅费 | 返回后 2 日内 | 每超1天不予报销 |

### 核销材料清单
| 材料类型 | 必须项 | 可选项 |
|---------|-------|-------|
| 发票 | ✅ 增值税专用发票 | — |
| 合同/协议 | ✅ 盖章生效版 | — |
| 审批单 | ✅ 完整审批流 | — |
| 成果证明 | ✅ 照片/数据/报告 | — |
| 验收记录 | ✅ 签字确认 | — |

### 核销异常处理
| 异常类型 | 处理流程 | 责任人 |
|---------|---------|-------|
| 材料不全 | 退回补全，重新计时 | 申请人 |
| 超预算 | 追加预算审批或拒批 | 审批人 |
| 超标准 | 按标准报销，超额自理 | 申请人 |
| 虚假报销 | 追回款项，通报批评 | 申请人+审核人 |

## 费效比分析模型

### 核心费效指标
| 指标 | 计算公式 | 健康标准 |
|------|---------|---------|
| 单线索成本 | 总广告费 / 总线索数 | ≤ 行业均值 |
| 单订单成本 | 总市场费 / 总订单数 | ≤ 品牌商务政策要求 |
| 费效比 | 总市场费 / 总销售额 | 0.5-2.0% |
| 投入产出比 | 增量销售额 / 增量市场费 | ≥ 3 |

### 费用效率评估矩阵
```
高费用效率
│
│  ★ 明星项目（高费效，持续投入）
│
│  ⚠️ 问题项目（低费效，需优化）
└─────────────────→ 费用规模
```

### 优化建议生成逻辑
```
IF 单线索成本 > 行业均值 1.2倍 THEN
  建议：优化投放渠道/调整出价策略/优化创意
IF 活动费占比 > 40% AND 费效比 < 1 THEN
  建议：减少低效活动，增加广告投放
IF 物料费人均成本 > 100元/月 AND 使用率 < 60% THEN
  建议：控制采购量，提高使用率
```

## 自动化监控规则

### 预算执行监控
| 监控点 | 触发条件 | 告警动作 |
|-------|---------|---------|
| 月度超支 | 执行率 > 120% | 飞书推送审批人+财务 |
| 季度失衡 | 单品牌费用占比 > 40% | 建议调整分配 |
| 年度预警 | 半年执行率 > 70% | 提醒控制节奏 |

### 核销进度监控
| 监控点 | 触发条件 | 告警动作 |
|-------|---------|---------|
| 逾期预警 | 距核销期限剩1天 | 飞书提醒申请人 |
| 逾期处理 | 超期1天 | 扣绩效分并升级提醒 |
| 严重逾期 | 超期7天 | 冻结该申请人后续报销 |

### 费效异常监控
| 监控点 | 触发条件 | 告警动作 |
|-------|---------|---------|
| 单线索成本异常 | > 行业均值1.5倍 | 专项分析报告 |
| 费效比恶化 | 连续2月下降 > 20% | 优化建议推送 |
| 投入产出比过低 | < 2 | 项目复盘会议 |
