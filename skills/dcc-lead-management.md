## 线索分级标准

| 等级 | 判定条件 | 响应要求 | 跟进频率 |
|------|----------|----------|----------|
| **H（高意向）** | 已明确车型+预算+购车时间，主动来电 | 15 分钟内 | 每日跟进 |
| **A（较高意向）** | 已确定品牌/车型，有比价行为 | 30 分钟内 | 每 2 天跟进 |
| **B（一般意向）** | 询价但未确定，处于信息收集阶段 | 1 小时内 | 每周跟进 |
| **C（低意向）** | 随便看看，无明确购车计划 | 4 小时内 | 每 2 周跟进 |

## 线索分配算法

```python
def assign_lead(lead, team):
    """
    线索分配策略：
    1. 按品牌分配给对应品牌组专员
    2. 同品牌组内按当前负荷轮转
    3. H 级线索优先分配给邀约到店率最高的专员
    4. 同一客户的历史跟进专员优先匹配
    """
    brand = lead.get('brand')
    level = lead.get('level')
    customer_id = lead.get('customer_id')

    # 查找历史跟进专员
    history_agent = find_history_agent(customer_id)
    if history_agent and history_agent.status == 'active':
        return history_agent

    # 品牌组匹配
    brand_team = get_brand_team(brand)

    # H级线索：加权分配（到店率高的专员权重更高）
    if level == 'H':
        return weighted_assign(brand_team, key='visit_rate')
    else:
        return round_robin_assign(brand_team, key='current_load')
```

## 线索跟进 SOP

### 首次联系（30 分钟内）

```
步骤1：自报家门（铠祺集团 XX 品牌 + 姓名）
步骤2：确认客户需求（"看到您在 XX 平台咨询了 XX 车型"）
步骤3：核心卖点介绍（不超过 3 个卖点）
步骤4：试驾邀约（"我们这周末有试驾体验活动"）
步骤5：添加微信 + 发送车型资料
步骤6：CRM 系统记录跟进备注
```

### 二次跟进（24 小时内）

```
步骤1：回顾上次沟通内容
步骤2：解答上次遗留疑问
步骤3：发送竞品对比资料（如客户在比价）
步骤4：再次邀约到店（用限时活动/现车为钩子）
步骤5：更新意向等级
```
