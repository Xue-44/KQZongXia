## 事故车维修全流程

```
1. 接车 → 记录事故信息 + 拍照（外观/损伤细节）
2. 报案 → 联系保险公司报案，获取报案号
3. 定损 → 保险公司查勘员到场/线上定损
4. 核价 → 确认维修项目和价格
5. 维修 → 钣金→喷漆→装配→质检
6. 交车 → 客户验车 + 签收 + 结算
7. 理赔 → 收集理赔资料，提交保险公司
8. 回款 → 跟踪保险公司打款
```
