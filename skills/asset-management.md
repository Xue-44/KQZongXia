# 资产管理技能

## 技能定位
本技能用于集团及下属门店固定资产的全生命周期管理，包括购置、登记、领用、调拨、维修、报废等环节，确保资产账实相符、价值清晰。

## 支持场景
| 场景 | 说明 | 频率 |
|------|------|------|
| 资产购置 | 新资产采购申请、验收、入库 | 按需 |
| 资产登记 | 贴标签、录台账、建卡片 | 实时 |
| 资产领用/调拨 | 员工领用、跨部门调拨 | 按需 |
| 资产盘点 | 全集团固定资产全面清点 | 月度/年度 |
| 资产维修 | 维修申请、费用审批、维修记录 | 按需 |
| 资产折旧 | 按月计提折旧，更新台账 | 月度 |
| 资产报废 | 报废申请、审批、处置 | 按需 |

## 资产分类

| 大类 | 细分类别 | 折旧年限 | 残值率 |
|------|---------|---------|--------|
| 电子设备 | 电脑、打印机、扫描仪、投影仪 | 3年 | 5% |
| 办公家具 | 桌椅、文件柜、沙发、会议桌 | 5年 | 5% |
| 车辆 | 公务车、试驾车、通勤车 | 5年 | 5% |
| 房屋建筑 | 展厅、办公区、维修车间 | 20年 | 5% |
| 其他设备 | 空调、监控、音响、充电桩 | 5年 | 5% |

## 折旧计算
- 方法: 直线法（平均年限法）
- 月折旧额 = (原值 - 残值) / (使用年限 × 12)
- 累计折旧 = 月折旧额 × 已使用月数
- 净值 = 原值 - 累计折旧

## 盘点流程
1. **盘点计划**: 明确盘点范围、时间、人员分工
2. **自盘**: 各使用部门先行自查
3. **复盘**: 行政中心组织交叉复盘
4. **差异处理**: 盘盈/盘亏分析原因并处理
5. **报告**: 出具盘点报告，报领导审批
6. **台账更新**: 根据盘点结果更新资产台账
