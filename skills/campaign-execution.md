# 市场活动展开技能 (Campaign Execution)

## 技能定位

为集团四品牌（广汽传祺、上汽奥迪、广汽昊铂、广汽埃安）提供市场活动全生命周期管理能力，覆盖策划、执行、检查、改进全流程，适配各品牌差异化策略。

---

## 1. 活动执行标准流程（PDCA 循环）

### 1.1 Plan — 策划阶段

#### 活动目标设定（SMART 原则）

| 维度 | 指标示例 | 数据来源 |
|------|---------|---------|
| 曝光量 | 活动总曝光人次 | 媒体/平台后台 |
| 留资/线索 | 有效线索数 | CRM/表单系统 |
| 转化率 | 线索→到店率、到店→成交率 | CRM |
| 传播度 | 社交媒体互动量（赞/评/转） | 各平台后台 |
| 满意度 | NPS/满意度评分 | 问卷调研 |

#### 预算模板

| 类别 | 明细 | 预估金额 | 实际金额 | 差异 |
|------|------|---------|---------|------|
| 场地费 | 租赁+搭建+拆除 | | | |
| 物料费 | 设计+制作+运输 | | | |
| 人员费 | 主持人/模特/摄影/礼仪 | | | |
| 媒体费 | 媒体车马费/发布费/KOL | | | |
| 餐饮费 | 茶歇/晚宴/工作餐 | | | |
| 礼品费 | 伴手礼/抽奖礼品 | | | |
| 技术费 | 灯光/音响/LED屏/直播 | | | |
| 应急预备 | 总预算的 10-15% | | | |
| **合计** | | **0** | **0** | **0** |

#### 时间表模板（倒推法）

```
活动日期 T-0：
T-45：内部立项审批
T-40：确定活动方案、场地选址
T-35：供应商招标/比价
T-30：确定供应商、签订合同
T-25：设计物料初稿
T-20：物料定稿、进入制作
T-15：媒体/KOL 邀约启动
T-10：物料到位检查、彩排第一次
T-7： 客户/嘉宾邀约截止、彩排第二次
T-3： 全场联排
T-1： 物料进场、搭建完成
T-0： 活动执行
T+1： 撤场
T+3： 效果数据汇总
T+7： 活动总结报告
```

#### 资源清单

- **场地资源**：场地位置图、容量、停车位、电力容量、网络带宽、消防通道
- **人员资源**：项目经理、主持人、摄影师、摄像师、礼仪、安保、志愿者
- **物料资源**：全部物料清单（详见 §2.4 物料清单模板）
- **技术资源**：LED屏、音响系统、灯光系统、直播设备、签到系统
- **媒体资源**：媒体名单、KOL名单、新闻稿、媒体资料包

#### 经验总结结构

1. **活动概况**：时间、地点、参与人数、品牌
2. **目标达成**：各项KPI完成情况
3. **亮点复盘**：做得好的3-5个点（可复用）
4. **问题复盘**：待改进的3-5个点（含原因分析）
5. **财务决算**：预算 vs 实际支出
6. **团队评价**：关键成员表现评价
7. **改进建议**：下次活动优化方向

## 2. 活动类型与执行要点

### 2.1 新车上市活动

| 环节 | 执行要点 | 关键产出物 |
|------|---------|-----------|
| 媒体邀请 | 汽车垂直媒体 + 生活方式媒体 + 区域媒体；提前 15 天发送邀请函，提前 3 天确认 | 媒体名单、邀请函 |
| 产品讲解 | 产品经理 15 分钟核心卖点讲解 + Q&A；准备产品手册和对比资料 | 讲解PPT、产品手册 |
| 试驾安排 | 路线规划（城市 + 高速 + 景区）、保险购买、安全须知签署 | 试驾路线图、试驾协议 |
| 新闻稿发布 | 活动当天发布通稿 + 各媒体定制稿；准备图文素材包 | 新闻通稿、素材包 |
| 直播/短视频 | 安排直播机位 + 短视频拍摄剪辑（活动当天出片） | 直播方案、短视频 |
| 车主证言 | 邀请首批车主/意见领袖现场分享 | 车主邀请函、发言稿 |

### 2.2 促销活动

| 环节 | 执行要点 | 关键产出物 |
|------|---------|-----------|
| 价格政策 | 明确优惠幅度、适用条件、截止时间（制造紧迫感） | 政策文件、价格牌 |
| 礼品准备 | 按预算采购，确保礼品充足+质量，准备备选方案 | 礼品清单、采购单 |
| 销售话术 | 统一接待话术、价格谈判策略、竞品对比话术 | 话术手册、FAQ |
| 展厅布置 | 促销氛围物料（条幅/气球/价格牌/礼品展示区） | 布置方案、物料清单 |
| 转化追踪 | 到店登记→试驾→下单→成交全链路追踪 | CRM追踪表 |
| 线上引流 | 广告投放 + 社交传播 + 电话邀约 | 投放素材、话术 |

### 2.3 品牌体验活动

| 环节 | 执行要点 | 关键产出物 |
|------|---------|-----------|
| 体验区设计 | 按品牌调性设计沉浸式体验区（科技/环保/豪华） | 体验区设计图 |
| 互动环节 | AR/VR体验、拍照打卡、互动游戏 | 互动方案、技术对接 |
| 拍照打卡 | 设计拍照打卡墙 + 即拍即得 + 社交分享引导 | 打卡墙设计、引导物料 |
| 社交媒体传播 | 设置话题标签 + UGC激励（打卡发朋友圈领礼品） | 传播方案、激励规则 |
| 数据采集 | 签到系统 + 行为数据埋点 | 数据采集方案 |
| 后续跟进 | 活动后 48 小时内发送感谢 + 试驾邀约 | 跟进话术、CRM标签 |

### 2.4 客户答谢活动

| 环节 | 执行要点 | 关键产出物 |
|------|---------|-----------|
| 嘉宾邀请 | 筛选 VIP 客户 + 意向客户；提前 14 天发出邀请，提前 3 天确认 | 嘉宾名单、邀请函 |
| 餐饮安排 | 按品牌调性选餐饮形式（私宴/自助/酒会）；照顾特殊饮食需求 | 菜单、餐饮合同 |
| 礼品发放 | 定制礼品（体现品牌元素）+ 礼盒包装 | 礼品清单 |
| 关系维护 | 销售顾问一对一陪同、活动后回访、定期问候 | 接待分工表、回访计划 |
| 氛围营造 | 音乐/灯光/布置营造温馨/高端氛围 | 氛围方案 |
| 增值环节 | 理财/美妆/品酒等非汽车类增值体验 | 合作方对接 |

## 4. 活动执行工具包

### 4.1 活动检查清单模板

详见 `scripts/campaign_tools.py` 中的 `generate_checklist()` 函数。生成物为 Markdown 格式，包含：

- 策划阶段检查项（15 项）
- 执行阶段检查项（20 项）
- 现场检查项（15 项）
- 收尾阶段检查项（10 项）

### 4.2 人员分工表模板

使用 `scripts/campaign_tools.py` 中的 `generate_raci_matrix()` 生成 RACI 矩阵 Excel 文件。

### 4.3 物料清单模板

使用 `scripts/campaign_tools.py` 中的 `generate_material_list()` 生成物料清单 Excel，包含字段：

- 物料名称、数量、规格、材质、设计稿链接、制作方、单价、总价、到位时间、当前状态

### 4.4 风险应对预案模板

使用 `scripts/campaign_tools.py` 中的 `generate_risk_matrix()` 生成风险评估矩阵：

- 风险项、概率（高/中/低）、影响（极高/高/中/低）、风险等级、应对措施、责任人、触发条件

### 4.5 效果评估报告模板

使用 `scripts/campaign_tools.py` 中的 `generate_evaluation_report()` 生成活动效果评估报告（PPTX/Excel）。

包含：活动概况、KPI达成、传播效果、线索效果、财务决算、亮点与问题、改进建议。
