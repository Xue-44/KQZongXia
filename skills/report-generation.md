# 报告生成技能 — 集团综合版

> **版本**: v2.0 | **更新日期**: 2026-05-23 | **适用 Agent**: 总虾 (xia-zong)
> **来源**: 市场虾 + 新媒体虾 合并去重

## 技能定位

为铠祺集团提供标准化、自动化、可视化的报告生成能力，覆盖日报/周报/月报/专项报告/年度报告全类型，支持PPTX和Excel双格式输出，面向市场部、新媒体部、销售部、售后部、财务部、客服部、行政部7个部门。

## 适用场景

- 各业务线数据日报/周报/月报自动生成
- 新车上市效果评估专项报告
- 促销活动ROI分析报告
- 新媒体运营报告（日报/周报/月报/投放复盘/爆款拆解）
- 年度总结与规划报告
- 管理层决策汇报材料
- 竞品深度分析报告
- 季度/年度运营总结

## 标准流程

1. **需求确认**：报告类型 → 时间范围 → 目标受众 → 交付格式
2. **数据准备**：从各数据源采集原始数据，清洗校验
3. **数据分析**：多维度交叉分析，发现趋势和异常
4. **结构组织**：按对应模板组织内容层次，提炼核心观点
5. **初稿撰写**：生成 Markdown 版本，数据驱动，结论有据
6. **可视化**：选择合适图表类型，按规范配色标注
7. **格式输出**：生成 PPTX / Excel / PDF，检查排版后交付

## 报告类型与模板

### 1. 日报（每日 09:30 前交付）

```
结构：
┌─────────────────────────────────┐
│ 数据日报 — 2026年XX月XX日       │
├─────────────────────────────────┤
│ 一、昨日核心数据概览             │
│   | 品牌 | 新增线索 | 到店量 | 成交量 | 线索成本 |
│   |------|---------|-------|-------|---------|
│   | 传祺 | XX     | XX    | XX    | ¥XX     |
│   | 奥迪 | XX     | XX    | XX    | ¥XX     |
│   | 昊铂 | XX     | XX    | XX    | ¥XX     |
│   | 埃安 | XX     | XX    | XX    | ¥XX     |
│   | 合计 | XX     | XX    | XX    | ¥XX     |
├─────────────────────────────────┤
│ 二、亮点内容（播放量 Top 3）     │
│ 三、重点渠道效果                 │
│ 四、异常预警（如有）              │
│ 五、今日执行提醒                 │
└─────────────────────────────────┘
```

**目标受众**：各部门内部团队
**篇幅**：一屏可读完，飞书卡片消息
**数据来源**：CRM系统 + 投放后台 + 销售日报

### 2. 周报（每周一 10:00 交付）

```
结构：
一、本周核心数据总览（环比变化）
   - 四品牌7日趋势折线图（线索/到店/成交）
   - 各品牌预算消耗与余额
   - 周目标达成率

二、各品牌表现拆解
  每个品牌包含：线索来源渠道分布、有效线索率、到店转化率、
  投放TOP3内容效果、爆款内容分析（Top 5）、低效内容诊断

三、竞品动态摘要
   - 本周竞品重大动作 / 促销政策变化 / 风险点

四、下周计划
   - 投放计划 / 重点活动预告 / 预算预估

五、待决策事项
```

**目标受众**：部门负责人 + 管理层
**篇幅**：Markdown 约 3000 字 + 8-12 个图表
**交付格式**：飞书文档 + PDF备份

### 3. 月报（每月5日前交付）

```
结构：
一、月度执行摘要（Executive Summary）
   - 核心结论（3条，一句话）
   - 核心指标达成仪表盘
   - 预算执行情况

二、各品牌深度分析
  每个品牌：线索/到店/成交趋势、渠道贡献度、线索成本、
  粉丝增长曲线、内容效果分布、投放ROI分析、团队产能

三、竞品月度对标分析
   - 市场份额变化 / 竞品评分雷达图 / 投入产出对比

四、专项活动复盘（如有）

五、行业趋势与机会洞察

六、下月策略建议（预算调整 / 渠道优化 / 内容策略调整）

七、附录（原始数据表 + 计算方法说明）
```

**目标受众**：管理层 + 集团汇报 + 品牌方
**篇幅**：PPTX 25-35页 + 配套Excel数据表
**交付格式**：PPTX + Excel

### 4. 管理层汇报PPT

```
结构（10-15页）：
1. 封面（报告主题 + 时间范围）
2. 核心结论（一页总览）
3-8. 各维度数据详析（每页一个核心观点 + 图表支撑）
9. 竞品对比
10. 问题与挑战
11. 下阶段计划
12. 需要支持的资源
```

### 5. 专项分析报告

```
结构：
- 背景与目标
- 数据概览
- 深度分析（多维度交叉）
- 核心发现（3-5条）
- 优化建议（可落地、有时限）
- 附录（原始数据 + 方法说明）
```

## 统一指标定义

| 指标 | 定义 | 计算方式 | 数据来源 |
|------|------|---------|---------|
| 线索量 | 所有渠道获取的有效客户联系方式 | 去重计数 | CRM系统 |
| 有效线索率 | 可接通且有意向的线索占比 | 有效线索/总线索 | CRM+DCC外呼 |
| 到店量 | 实际到店客户数 | 展厅登记计数 | 展厅接待系统 |
| 到店率 | 线索转化为到店的比例 | 到店量/线索量 | CRM系统 |
| 试驾量 | 实际完成试驾的客户数 | 试驾登记计数 | 试驾登记表 |
| 成交量 | 实际完成签约的订单数 | 签约计数 | 销售系统 |
| 线索成本（CPL） | 每条线索的平均获取成本 | 总投放费用/线索量 | 投放后台+CRM |
| 成交成本 | 每台成交的平均市场成本 | 总市场费用/成交量 | 财务+销售系统 |
| ROI | 市场投入产出比 | (成交毛利-市场费用)/市场费用 | 财务系统 |

## 数据可视化规范

### 图表类型选择

| 数据类型 | 推荐图表 | 示例场景 |
|---------|---------|---------|
| 时间趋势 | 折线图 | 线索量/到店量/成交量月度变化 |
| 横向对比 | 柱状图/条形图 | 各品牌线索成本对比 |
| 占比分布 | 饼图/环形图/堆叠柱状图 | 渠道贡献度分布 |
| 相关性 | 散点图/气泡图 | 投放费用 vs 线索量关系 |
| 转化漏斗 | 漏斗图 | 线索->到店->试驾->成交 |
| 多维度对比 | 雷达图 | 竞品综合评分对比 |
| 时间x指标 | 热力图 | 各时段各渠道效果矩阵 |
| 进度 | 仪表盘/进度条 | KPI达成率展示 |

### 图表标注规范

- 所有图表标注数据来源和时间范围（左下角，9pt灰色字体）
- 环比/同比变化：增长=绿色箭头+数字，下降=红色箭头+数字
- 异常数据点：红色虚线框 + 简短注释
- 坐标轴：清晰标注单位和刻度
- 数据标签：关键数据点直接标注数值

## PPT 生成规范

### 页面布局
- **比例**：16:9 宽屏
- **区域划分**：标题区（顶部15%）+ 内容区（中部75%）+ 页码/来源（底部10%）
- **核心原则**：每页1个核心观点 + 图表支撑，图表占面积60%以上
- **信息密度**：每页文字不超过80字

### 文字规范

| 层级 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 封面主标题 | 36pt | Bold | 品牌色 |
| 页面标题 | 28pt | Bold | #1A237E（总虾深海蓝） |
| 二级标题 | 22pt | Bold | #1F2937 |
| 正文 | 18pt | Regular | #374151 |
| 数据标注/注释 | 14pt | Regular | #6B7280 |
| 图表标签 | 12pt | Regular | #6B7280 |

## 报告质量检查清单

### 数据质量
- [ ] 所有数据来源清晰标注，可追溯
- [ ] 数据计算无错误（抽查3-5个关键数值）
- [ ] 环比/同比方向正确（上升=绿、下降=红）
- [ ] 数据时间范围与报告标题一致

### 内容质量
- [ ] 核心结论有充分数据支撑
- [ ] 建议可落地、有时限、有责任人
- [ ] 竞品数据真实可靠（非猜测）
- [ ] 异常数据有解释说明

### 可视化质量
- [ ] 图表类型选择恰当
- [ ] 配色符合品牌规范
- [ ] 图表标注完整（标题+单位+来源+时间）

### 格式质量
- [ ] PPTX 16:9比例正确
- [ ] 字体嵌入或使用系统通用字体
- [ ] 文件命名规范（类型_品牌_时间范围_版本）
