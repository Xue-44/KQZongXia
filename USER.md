# 用户使用指南 — 总虾

## 总虾是谁？

**总虾**是铠祺集团运营总经理的AI助手，整合了7个部门的专业能力：

1. **市场虾**：市场费用核销、预算管控、活动策划
2. **新媒体虾**：内容运营、短视频制作、社交媒体管理
3. **销售虾**：销售漏斗、库存管理、KPI看板
4. **售后虾**：保险理赔、维保管理、配件库存
5. **财务虾**：预算控制、财务报告、返利管理
6. **客服虾**：投诉处理、客户满意度、电话脚本
7. **行政虾**：资产管理、文档管理、会议管理

## 如何与总虾对话？

### 1. 基本对话格式
```
[部门] + [任务] + [具体要求]
示例：
"市场部，帮我生成一份5月份的费用核销报告"
"新媒体部，策划一个传祺GS8的抖音短视频"
"销售部，查看一下传祺M8的库存情况"
```

### 2. 多部门协同任务
```
"我需要一个完整的活动方案，包括市场策划、新媒体宣传、销售转化"
"帮我分析一下奥迪A7L的销售问题，从市场、销售、售后多个角度"
```

### 3. 数据查询任务
```
"查看传祺品牌4月份的线索数据"
"对比一下昊铂和埃安5月份的投放效果"
"生成一份集团整体的经营分析报告"
```

## 文件输出格式

### 1. 报告类输出
- **日报**：飞书卡片消息，一屏可读完
- **周报**：Markdown文档 + PDF备份
- **月报**：PPTX演示文稿 + Excel数据表
- **专项报告**：PPTX深度分析 + Markdown详细说明

### 2. 数据类输出
- **Excel表格**：费用台账、预算执行表、销售报表
- **CSV文件**：原始数据导出
- **JSON文件**：结构化数据
- **图表文件**：PNG/JPG格式图表

### 3. 设计类输出
- **图片文件**：海报、Banner、社交媒体图片
- **视频脚本**：Markdown格式脚本
- **设计稿**：PSD/AI源文件（需人工处理）

## 文件管理

### 1. 文件位置
- **技能文件**：`skills/` 目录
- **工具脚本**：`scripts/` 目录
- **品牌数据**：`brands/` 目录
- **记忆存储**：`memory/` 目录
- **模板文件**：`templates/` 目录

### 2. 文件命名规范
```
报告类：{类型}_{品牌}_{时间范围}_{版本}.{格式}
示例：monthly_report_trumpchi_2026-05_v1.pptx

数据类：{数据类型}_{品牌}_{日期}_{版本}.{格式}
示例：leads_data_audi_2026-05-23_v1.xlsx

设计类：{设计类型}_{品牌}_{用途}_{版本}.{格式}
示例：poster_trumpchi_promotion_v1.jpg
```

### 3. 文件版本管理
- v1：初版
- v2：修改版
- v3：终版
- final：最终版
- draft：草稿版

## 问题排查

### 1. 常见问题
```
Q: 总虾没有回应怎么办？
A: 检查网络连接，确认指令格式正确

Q: 数据查询结果不对怎么办？
A: 确认时间范围、品牌、指标是否正确

Q: 文件生成失败怎么办？
A: 检查文件路径权限，确认数据格式正确

Q: 自动化任务没有执行怎么办？
A: 检查系统时间，确认任务配置正确
```

### 2. 错误信息解读
```
"参数错误"：检查指令格式，确认必填参数
"文件不存在"：检查文件路径，确认文件存在
"权限不足"：检查文件权限，确认有访问权限
"数据错误"：检查数据格式，确认数据正确
"系统错误"：联系系统管理员
```

### 3. 联系支持
- **技术问题**：联系IT部门
- **业务问题**：联系相关部门负责人
- **数据问题**：联系数据分析师
- **系统问题**：联系系统管理员
