# 客户满意度管理

## 技能名称
客户满意度调研与提升策略

## 适用场景

| 场景 | 触发词 | 输出 |
|------|--------|------|
| 满意度调研 | "满意度""回访""NPS""打分" | 调研问卷 + 分析报告 |
| 回访排班 | "回访计划""排班""谁回访" | 回访排班表 |
| 满意度分析 | "满意度报告""短板""哪里差" | 分析报告 + 改进方案 |
| NPS 计算 | "NPS""净推荐值" | NPS 评分 + 分群分析 |

## 回访体系

### 回访节点

| 节点 | 时间 | 方式 | 重点问题 |
|------|------|------|----------|
| 首次到店回访 | 到店后 24 小时 | 电话 | 展厅体验、销售接待、试驾感受 |
| 交车回访 | 交车后 3 天 | 电话 | 交车体验、车辆讲解、手续办理 |
| 首保回访 | 首保提醒前 7 天 | 电话+短信 | 用车感受、问题收集、首保预约 |
| 满意度季度调研 | 每季度末 | 电话 | NPS、整体满意度、推荐意愿 |
| 流失客户回访 | 战败后 7 天 | 电话 | 战败原因、竞品选择、改进建议 |

### 满意度评分标准（10 分制）

| 分数 | 等级 | 定义 | 处理方式 |
|------|------|------|----------|
| 9-10 | 推荐者 | 非常满意，愿意推荐 | 记录为成功案例，邀请转介绍 |
| 7-8 | 被动者 | 基本满意，无明确不满 | 正常维护，定期关怀 |
| 5-6 | 贬损者 | 有不满情绪，可能投诉 | 48 小时内二次跟进，解决问题 |
| 1-4 | 高流失风险 | 极度不满，可能公开投诉 | 24 小时内由主管亲自跟进 |

## 满意度提升策略库

### 低频原因 → 高频对策

| 问题 | 短期对策（本周） | 中期对策（本月） | 长期对策（本季度） |
|------|-----------------|-----------------|-------------------|
| 销售态度差 | 当事销售停岗培训 | 全员服务礼仪培训 | 建立神秘访客制度 |
| 交车延迟 | 每日更新交车进度表 | 优化交车排期流程 | 与主机厂沟通产能 |
| 价格争议 | 提供额外精品礼包 | 统一报价体系 | 透明化价格公示 |
| 售后慢 | 开通绿色通道 | 增加售后接待人员 | 预约体系数字化 |
| 金融不满 | 更换金融方案 | 引入多家金融机构 | 定制金融产品 |

## 月度满意度报告结构

```
1. 核心指标速览（NPS / 满意度均分 / 推荐率 / 贬损率）
2. 品牌维度对比
3. 销售顾问排名（Top 3 & Bottom 3）
4. 贬损原因 Pareto 分析
5. 重点改进事项（Top 3 问题 + 行动计划）
6. 客户原声摘录（正面 3 条 + 负面 3 条）
7. 下月提升目标
```
