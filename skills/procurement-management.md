# 采购管理技能

## 技能定位
本技能用于集团行政类采购的全流程管理，包括需求收集、询价比价、审批、合同签订、验收付款等环节，确保采购合规、成本可控。

## 支持场景
| 场景 | 说明 | 频率 |
|------|------|------|
| 日常采购 | 办公用品、耗材、饮用水等 | 每周/每月 |
| 固定资产采购 | 电脑、打印机、家具等 | 按需 |
| 服务类采购 | 保洁、安保、绿化、维修 | 年度/按需 |
| 活动物资采购 | 车展、开业、庆典等物资 | 按需 |
| 紧急采购 | 突发需求的快速采购通道 | 按需 |

## 采购审批权限

| 金额范围 | 审批层级 |
|---------|---------|
| ≤ 500元 | 部门负责人 |
| 501-2000元 | 行政总监 |
| 2001-5000元 | 行政总监 + 财务审核 |
| 5001-10000元 | 行政总监 + 财务审核 + 副总经理 |
| > 10000元 | 行政总监 + 财务审核 + 副总经理 + 总经理 |

## 采购流程

1. **需求提出**: 填写《采购申请单》
2. **询价比价**: 至少3家供应商比价
3. **审批**: 按金额走审批流程
4. **合同签订**: 大额采购需签采购合同
5. **下单**: 向选定供应商下达采购订单
6. **验收**: 收货验收，核对数量和质量
7. **入库/领用**: 登记入库或直接领用
8. **付款**: 凭发票、验收单申请付款
9. **归档**: 采购全流程文件归档

## 比价原则
- 同等质量比价格
- 同等价格比质量
- 同质同价比服务
- 优先选择长期合作供应商
- 年度采购可签框架协议
