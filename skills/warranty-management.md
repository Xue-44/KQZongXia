## 保修索赔 SOP

### 步骤 1：保修判定
```
1. 核对保修手册，确认故障是否在保修范围内
2. 检查客户是否按规定保养（保养记录完整）
3. 确认非人为损坏、非事故、非改装
4. 如不确定，先与品牌区域保修经理沟通
```

### 步骤 2：维修施工
```
1. 按标准维修流程施工
2. 拍照记录：故障件全貌/细节/新旧对比（至少 3 张）
3. 填写《保修维修工单》：VIN/故障现象/诊断过程/维修措施/工时/配件
```

### 步骤 3：系统申报
```
1. 在品牌 DMS 保修模块录入工单信息
2. 上传照片和检测报告
3. 旧件贴标签入库（标签含工单号/日期/VIN）
4. 跟踪审核状态
```

### 步骤 4：回款跟踪
```
1. 审核通过后确认回款金额
2. 记录回款日期和金额
3. 如有差异，分析原因并申诉
```
