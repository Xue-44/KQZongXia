# 电话邀约话术

## 技能名称
标准化电话邀约与回访话术库

## 适用场景

| 场景 | 触发词 | 输出 |
|------|--------|------|
| 首电邀约 | "首电""首次联系""刚咨询" | 邀约话术脚本 |
| 试驾邀约 | "试驾""试乘""体验" | 试驾邀约话术 |
| 活动邀约 | "团购""活动""周末活动" | 活动邀约话术 |
| 逼单话术 | "逼单""冲量""月底" | 逼单话术 |
| 战败挽留 | "战败""选了竞品""不考虑了" | 挽留话术 |
| 回访话术 | "回访""满意度""用车感受" | 回访话术 |
| 沉睡唤醒 | "唤醒""很久没联系""老客户" | 唤醒话术 |

## 话术结构框架（AIDA 模型）

```
A - Attention（引起注意）：品牌+姓名+切入点
I - Interest（激发兴趣）：核心卖点匹配客户需求
D - Desire（刺激欲望）：限时/限量/活动制造紧迫感
A - Action（促成行动）：明确时间地点，降低到店门槛
```

## 场景话术

### 1. 首电邀约（网络线索）

```
【背景】客户在 XX 平台咨询了 XX 车型

话术脚本：
"您好，请问是 X 先生/女士吗？我是宁夏铠祺 [品牌] 的客服专员 [姓名]。
看到您在 [懂车帝/汽车之家/抖音] 上关注了我们 [车型名称]，
想了解一下您大概什么时候方便到店看看实车？我们展厅有现车可以试驾。"

→ 如果客户说"先了解价格"：
"价格方面肯定给您最优方案。不过买车还是得亲自感受一下，
您这周末方便吗？我带您试驾一圈，顺便给您算一份详细落地价。"
```

### 2. 试驾邀约

```
话术脚本：
"X 先生/女士，我们这个周末刚好有一场 [车型] 深度试驾会，
不是那种绕一圈就结束的，整整 30 分钟的深度体验，城市道路+高速路段都覆盖。
现在报名还送一份品牌定制礼品。您周六上午还是下午方便？"
```

### 3. 活动邀约（团购/周年庆）

```
话术脚本：
"X 先生/女士，告诉您一个好消息！铠祺集团这周末有周年庆团购活动，
[车型] 这次优惠力度是今年最大的，最高综合优惠 [金额] 元，
再送 [赠品/保养/保险]。但名额只有 [数量] 个，现在已经有 [已报名] 组客户报名了。
我把您的名字记上去？"
```

### 4. 逼单话术（月底冲量）

```
话术脚本：
"X 先生/女士，说实话，月底了，我们差 [数量] 台就能拿到厂家的季度返利，
这台车我们真的是亏本在冲量。您之前看中的那台 [车型] 还停在展厅，
如果能在这两天定下来，我帮您跟经理申请一个特殊价格，这机会真不是每个月都有的。"

→ 必须真实可信，不能虚假承诺。
```

### 5. 战败挽留

```
话术脚本：
"X 先生/女士，听说您最终选了 [竞品车型]，恭喜您提车！
我打这通电话不是来劝您的，就是好奇想问一下，
您最终没选我们的 [车型]，主要是因为哪方面没有满足您的需求？"

→ 记录战败原因后：
"谢谢您的坦诚反馈，这对我们非常有价值。
如果以后朋友有购车需求，麻烦您帮忙推荐一下。
或者您自己以后换车，也可以随时找我，祝您用车愉快。"
```

### 6. 交车后回访

```
【时机】交车后第 3 天

话术脚本：
"X 先生/女士，我是铠祺 [品牌] 客服专员 [姓名]。
恭喜您这几天喜提爱车！开起来感觉怎么样？有没有什么功能不太熟悉的？
另外，您提车当天我们销售顾问的服务您满意吗？我这边做一个简单的满意度记录。"

→ 如果评分 ≥ 9 分：
"非常感谢您的认可！后续用车有任何问题随时找我。"

→ 如果评分 ≤ 7 分：
"感谢您的反馈，您说的这个问题我会亲自跟进，稍后给您回复处理方案。"
```

## 电话沟通黄金法则

### 时间选择

| 客户类型 | 最佳通话时段 | 避免时段 |
|----------|-------------|----------|
| 上班族 | 12:00-13:30 / 18:30-20:00 | 9:00-11:30（会议密集） |
| 个体户/自由职业 | 10:00-11:30 / 14:00-16:00 | 8:00 前 / 21:00 后 |
| 网约车司机 | 11:00-13:00（充电/休息） | 早晚高峰 |
| 企业主/高管 | 10:30-11:00 / 15:00-16:00 | 周一上午 |

### 通话节奏

| 要素 | 标准 |
|------|------|
| 通话时长 | 首次邀约 3-5 分钟，回访 2-3 分钟 |
| 语速 | 160-180 字/分钟 |
| 打断率 | 0 次（让客户先说完） |
| 提问占比 | 40%（多问少说） |
| 确认次数 | ≥ 2 次（需求确认 + 时间确认） |
