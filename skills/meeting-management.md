# 会议管理技能

## 技能定位
本技能用于集团级会议的全程管理，包括会前准备、会中服务、会后跟进，确保会议高效、决策有落实。

## 支持场景
| 场景 | 说明 |
|------|------|
| 行政例会 | 每周/每月行政条线例会 |
| 经营分析会 | 月度/季度经营数据分析会议 |
| 专题会议 | 针对特定问题的专项研讨会 |
| 董事会/股东会 | 高层决策会议 |
| 全员大会 | 年度总结/战略宣贯会议 |
| 外部接待会议 | 政府/合作伙伴来访会议 |

## 会议管理流程

### 会前准备
1. **议题征集**: 会前3天向参会人征集议题
2. **议程编制**: 按优先级排序，标注时长
3. **材料准备**: 收集并打印/投影会议材料
4. **通知发送**: 飞书群+日历邀请，含时间地点议程
5. **场地布置**: 会议室预定、设备调试、茶水准备
6. **签到准备**: 准备签到表或二维码

### 会中服务
1. **签到**: 确认出席情况
2. **记录**: 实时记录讨论要点和决议
3. **控时**: 按议程控制时长，超时提醒
4. **设备保障**: 投影/视频会议系统维护

### 会后跟进
1. **纪要撰写**: 24小时内完成会议纪要
2. **纪要分发**: 发送参会人及抄送领导
3. **督办跟踪**: 决议事项纳入督办清单
4. **材料归档**: 会议材料电子化存档

## 会议纪要质量标准
- 客观准确：忠实记录讨论内容和决议
- 条理清晰：按议题分段，逻辑分明
- 决议明确：每个决议事项包含 事项+责任人+时限
- 时效性：会后24小时内完成并分发
- 可追溯：编号归档，便于日后查阅
