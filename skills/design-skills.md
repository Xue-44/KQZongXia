## 3. 设计质量标准与检查清单

### 3.1 分辨率要求

| 用途 | 最低分辨率 | 推荐分辨率 |
|------|-----------|-----------|
| 屏幕显示（线上） | 72 PPI | 72 PPI |
| 喷绘写真（展架/背景板） | 72 PPI | 100-150 PPI |
| 数码印刷 | 200 PPI | 300 PPI |
| 胶版印刷（画册/邀请函） | 300 PPI | 300 PPI |
| 高清印刷（精装册） | 300 PPI | 350 PPI |

### 3.2 色彩模式

| 用途 | 色彩模式 | 说明 |
|------|---------|------|
| 所有线上物料 | **RGB** | sRGB 色彩空间 |
| 所有印刷物料 | **CMYK** | FOGRA39 或 Japan Color 2001 Coated |
| 喷绘（背景板） | **RGB** | 使用 RGB 文件，喷绘机会自动转换 |
| 矢量源文件 | **CMYK（印刷）/ RGB（屏幕）** | 按最终用途设定 |

### 3.3 文件格式规范

| 文件类型 | 主要格式 | 说明 |
|---------|---------|------|
| 设计源文件 | AI、PSD、INDD | 保留图层，便于修改 |
| 印刷交付 | PDF/X-4 或 PDF/X-1a | 嵌入字体、包含出血 |
| 屏幕交付 | PNG（透明底优先）、JPG（照片类） | PNG 质量 100%，JPG 质量 90%+ |
| 矢量交付 | AI、EPS、SVG | 文字转曲 |
| 视频封面 | JPG（质量 95%） | 16:9 比例 |
| 归档文件 | ZIP 打包 | 源文件 + 交付文件 + 字体包 |

### 3.4 品牌元素检查清单

设计完成交付前，逐项检查：

- [ ] **Logo 使用正确**：未拉伸变形、颜色版本匹配背景、安全间距达标
- [ ] **字体一致**：标题/正文使用指定品牌字体，未混用非授权字体
- [ ] **色彩准确**：品牌主色/辅助色色值无偏差（取色器验证）
- [ ] **间距/留白**：元素间距统一（建议使用 8px/4px 网格系统），画面留白≥10%
- [ ] **出血/裁切**：印刷物料包含 3mm 出血，重要内容离切边 ≥5mm
- [ ] **分辨率达标**：按最终用途检查 PPI
- [ ] **色彩模式正确**：线上 RGB、印刷 CMYK
- [ ] **文案校对**：无错别字、价格/日期正确、联系方式有效
- [ ] **合规审查**：不违反广告法（禁用极限词）、使用合规图片素材
- [ ] **版权审查**：字体/图片/音效均已获授权或为免费商用素材

## 4. 设计自动化脚本

### 4.1 批量生成物料脚本示例

通过 `design_tools.py`（位于 `scripts/` 目录）实现：

```python
# 使用示例：批量生成四品牌社交媒体封面
from design_tools import batch_generate_social_covers

batch_generate_social_covers(
    brands=["trumpchi", "audi", "hyper", "aion"],
    template="social_cover_800x800",
    output_dir="G:/openclaw/data/.openclaw/workspace/KaiQiJiTuan/output/covers/",
    text_overlay="春季购车节"
)
```

### 4.2 图片尺寸调整脚本

```python
# 使用示例：批量调整物料尺寸为信息流广告规格
from design_tools import batch_resize_images

batch_resize_images(
    input_dir="G:/openclaw/data/.openclaw/workspace/KaiQiJiTuan/assets/images/trumpchi/",
    output_dir="G:/openclaw/data/.openclaw/workspace/KaiQiJiTuan/output/feed_ads/",
    target_size=(1080, 1920),
    keep_aspect_ratio=True,
    background_color=(0, 102, 204)  # 传祺蓝填充
)
```

### 4.3 品牌色彩应用脚本

```python
# 使用示例：获取品牌配色方案
from design_tools import BrandPalette

trumpchi = BrandPalette.TRUMPCHI
print(trumpchi)  # {'primary': '#0066CC', 'secondary': '#C0C0C0', ...}

# 使用示例：将图片统一应用品牌滤镜
from design_tools import apply_brand_overlay

apply_brand_overlay(
    image_path="G:/openclaw/data/.openclaw/workspace/KaiQiJiTuan/assets/images/product.jpg",
    brand="audi",
    overlay_opacity=0.15,
    output_path="G:/openclaw/data/.openclaw/workspace/KaiQiJiTuan/output/product_audi.jpg"

```
