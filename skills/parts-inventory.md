# 配件库存管理

## 技能名称
配件计划、采购、库存、盘点与呆滞件处置

## 适用场景

| 场景 | 触发词 | 输出 |
|------|--------|------|
| 库存查询 | "库存""还有几个""查配件" | 库存数据 |
| 订货建议 | "该订货了""补货""安全库存" | 建议订货清单 |
| 呆滞件分析 | "呆滞""太久没动""积压" | 呆滞件清单 + 处置建议 |
| 盘点管理 | "盘点""差异""对不上" | 盘点计划/差异分析 |
| 配件报价 | "多少钱""价格" | 配件价格 |

## 配件 ABC 分类

| 分类 | 定义 | 管理策略 | 盘点频率 |
|------|------|----------|----------|
| **A 类** | 占库存金额 70%、数量 10%（高价值核心件） | 每日关注、精确计算安全库存 | 月度 |
| **B 类** | 占库存金额 20%、数量 30%（常用保养件） | 周度关注、设置 min-max 库存 | 月度 |
| **C 类** | 占库存金额 10%、数量 60%（低值易耗件） | 月度关注、经济批量订货 | 季度 |

## 安全库存计算

```
安全库存 = 日均用量 × 采购周期 × 安全系数

各品牌配件采购周期：
- 广汽传祺：3-5 天（国产件为主）
- 上汽奥迪：7-14 天（进口件多）
- 广汽昊铂：5-10 天
- 广汽埃安：3-7 天

安全系数建议：
- A 类件：1.5（高性能车/稀有件）
- B 类件：1.3
- C 类件：1.2
```

## 配件订货决策矩阵

| 库存状态 | 采购周期短 | 采购周期长 |
|----------|-----------|-----------|
| 低于安全库存 | **立即订货** | **立即订货（加急）** |
| 接近安全库存 | 纳入周度订货计划 | 纳入周度订货计划 |
| 远高于安全库存 | 暂停订货 | 暂停订货 |

## 呆滞件处置策略

| 呆滞天数 | 处置策略 | 预期回收比例 |
|----------|----------|-------------|
| 90-180 天 | 主动推荐给进厂客户，优先使用 | 80-100% |
| 180-360 天 | 与供应商协商退货/换货 | 50-80% |
| > 360 天 | 折价处理 / 调拨给有需求门店 / 报废 | 10-50% |
