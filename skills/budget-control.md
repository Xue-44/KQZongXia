# 预算管理

## 技能名称
全面预算编制、执行监控与偏差分析

## 适用场景

| 场景 | 触发词 | 输出 |
|------|--------|------|
| 预算编制 | "做预算""编制预算""明年预算" | 预算方案 |
| 预算执行 | "预算用了多少""预算进度""还剩多少" | 预算执行表 |
| 偏差分析 | "预算偏差""超预算""为什么超了" | 偏差分析报告 |
| 预算调整 | "调整预算""追加预算""预算不够" | 调整方案 |

```

├── 收入预算

│   ├── 水平业务收入（金融/保险/精品/上牌）
│   └── 返利收入（按品牌/类型）

```

## 预算编制流程

```
Step 1: 基数确定（11月初）
  → 当年 1-10 月实际 + 11-12 月预测 = 当年基数
  → 考虑下年市场增长预期、主机厂销量目标

Step 2: 业务部门申报（11月中旬）
  → 销售部报销量目标 + 广宣计划
  → 售后部报产值目标 + 设备需求
  → 行政人事报人员编制 + 薪酬预算
  → 各门店报运营费用

Step 3: 财务汇总初审（11月下旬）
  → 收入预算合理性校验（对比历史/行业）
  → 费用预算压缩空间分析
  → 利润目标达成测算

Step 4: 集团审批（12月初）
  → 预算汇报 PPT（财务虾主讲）
  → 集团总经理审批

Step 5: 分解下发（12月中旬）
  → 年度 → 季度 → 月度
  → 部门 → 门店 → 科目
```

## 偏差分析框架

### 三级预警机制

| 预警 | 触发条件 | 响应 |
|------|----------|------|
| 🟢 正常 | 累计达成率 90%-105% | 无需额外动作 |
| 🟡 关注 | 累计达成率 80%-90% 或 105%-115% | 部门说明原因，财务持续跟踪 |
| 🔴 严重 | 累计达成率 < 80% 或 > 115% | 部门提交书面分析，财务虾审核，总经理特批 |

### 偏差根因分析

| 偏差方向 | 可能原因 | 处理建议 |
|----------|----------|----------|
| 收入低于预算 | 市场下行 / 竞品降价 / 车型切换 | 评估是否需要调减全年目标 |
| 收入高于预算 | 爆款车型 / 政策利好 / 大客户订单 | 关注可持续性，避免过度乐观 |
| 费用超预算 | 活动追加 / 人员超编 / 单价上涨 | 冻结非必要支出，超预算部分走特批 |
| 费用低于预算 | 活动取消 / 人员未到位 / 采购延迟 | 评估是否影响业务，释放预算空间 |
